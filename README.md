# Automatic Core Sample Logging using Machine Learning

## Overview
This project is an AI/ML-based geological core sample logging system that automates the identification and classification of rock core images using image processing and deep learning techniques.

The model analyzes uploaded core sample images and predicts geological categories with visualization support.

---

## Features
- Core sample image classification
- Geological texture analysis
- Image preprocessing using OpenCV
- CNN-based deep learning model

---

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## Project Workflow

1. Upload core sample image
2. Image preprocessing
3. Feature extraction
4. CNN model prediction
5. Mineral/rock classification
6. Visualization and report generation
